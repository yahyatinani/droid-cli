======
Fast, interactive CLI for Android projects. Scaffold Compose apps instantly with a single binary. Includes Gradle wrapper and proper structureâ€”no Bloated Studio needed.

Installation
--
Binary
Download the latest executable for your platform from the Releases page.

Usage
--
Run the executable!
The tool will perform an environment check and prompt you for App Name, Package Name.. You will eventually have a project at the current directory.

Building from Source
--
